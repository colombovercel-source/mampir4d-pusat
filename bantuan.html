PERTANYAAN YANG SERING DIAJUKAN (FAQ)
Bagaimana cara klaim Mystery Box MAMPIR4D? ▼
Caranya sangat mudah! Anda cukup masuk ke halaman utama Mystery Box, ketik User ID terdaftar Anda pada kolom yang disediakan, lalu klik tombol "BUKA MISTERI BOX". Sistem akan mengundi hadiah keberuntungan Anda secara instan.
Mengapa muncul pesan ID sudah pernah klaim? ▼
Setiap User ID resmi hanya memiliki kesempatan melakukan klaim sebanyak 1 kali per hari atau sesuai periode event aktif yang ditentukan. Jika muncul notifikasi tersebut, silakan coba kembali pada periode event berikutnya.
Apa yang harus dilakukan setelah memenangkan hadiah? ▼
Segera ambil screenshot (tangkapan layar) bukti pop-up kemenangan Anda yang menampilkan nominal hadiah dan nama User ID Anda. Berikan bukti tersebut langsung kepada Admin Telegram atau Customer Service kami melalui tombol di atas untuk proses pencairan saldo.
Apakah event klaim ini dipungut biaya? ▼
Tidak sama sekali! Seluruh rangkaian event Mystery Box di website MAMPIR4D bersifat 100% gratis untuk mengapresiasi para member setia kami. Waspadalah terhadap segala bentuk penipuan yang mengatasnamakan pihak kami.
